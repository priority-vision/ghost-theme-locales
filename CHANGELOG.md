# Changelog

## [0.0.6] - Nov 28, 2025

- added types
- export main function `extractTranslatableStrings` to use in 3rd-party libs

## [0.0.4] - Dec 6, 2024

- added support for tilde symbol in brackets like `{{t "Hello"~}}` or `{{~t "World"~}}`

## [0.0.3] - Oct 25, 2024

- added support for nested strings like `(t "Hello {tag}" tag="World")`

## [0.0.2] - Oct 16, 2024

- initial release
